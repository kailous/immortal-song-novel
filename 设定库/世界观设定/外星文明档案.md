# 设定库：外星文明档案 · 塔伦人

本文件记录入侵地球的外星文明的完整背景设定。所有内容均为已确认的世界观事实，后续创作不得与之矛盾。

## 二、 生理与形态

**起源**：纯碳基生物，进化路线与人类高度相似，起点几乎一致。

**现状**：大规模自愿义体化改造后，个体形态差异极大：

- **半身义体**：保留部分生物躯体，四肢或器官替换为硅基机械结构，最为常见
- **全身义体**：生物脑保留，躯体全部机械化
- **电子脑**：生物脑被硅基计算核心取代，配合量子纠缠神经网络运作；高层及军事精英多为此类

**外形**：双足直立，双侧对称，头部集中感知器官。与人类最显著的差异在于四臂结构与体型：

| 部位 | 描述 |
|------|------|
| **肩部大臂（×2）** | 位于肩膀，与人类手臂位置相同，但肌肉极度发达；主要负责力量性动作——战斗、攀爬、搬运 |
| **腰腹小臂（×2）** | 从腰腹两侧延伸，体型明显小于肩臂，但灵活度更高；专职精细操作——工具使用、技术作业 |
| **整体体型** | 类似黑猩猩的致密紧凑肌肉结构，无肌肉抑制素，单位体重力量远超人类；高基础代谢，天生高能耗体质 |
| **义体化分布** | 肩臂常被替换为重型机械义肢；腰腹小臂常集成精密工具或传感器 |

**高能耗体质的文明含义**：无肌肉抑制素的四臂体型使塔伦人的基础代谢需求远高于人类，能源危机的生物学根源早于任何技术选择——义体化与戴森球只是在一个天生高能耗物种上不断叠加消耗。

> 第一章描述的"碳硅混合体，神经系统靠量子纠缠维系"即指电子脑配合义体化的标准军事个体，并非生物进化结果。

## 三、 文明发展弧线

| 阶段 | 内容 |
|------|------|
| **诞生期** | 纯碳基，进化与文明发展历程与人类几乎同构：渔猎→农业→工业→信息革命 |
| **义体化时代** | 硅基改造技术成熟，个体自愿选择义体化；电子脑出现；改造逐渐普及，成为主流生活方式 |
| **星系扩张期** | 向伽南星系内扩张，大规模采掘行星与小行星物质资源；消耗进入指数增长 |
| **戴森球时代** | 围绕洛尔及周边恒星建造戴森球，几乎完全截获恒星能源输出；看似解决了能源问题，实为将危机规模放大 |
| **衰竭期** | 塔伦星系属老年椭圆星系，恒星本就暗淡，戴森球产出递减；行星物质已被大规模拆解用于建造；义体化人口的能源维护需求持续增长；消耗速度彻底超过宇宙补充速度 |
| **危机期** | 能源债务不可逆，文明进入灭绝倒计时；向宇宙发出求救信号 |
| **内战→跃迁** | 主战派政变胜利，动员全星最后资源完成跨星系跃迁 |

## 四、 资源危机的本质

**根本矛盾**：能源利用率始终是瓶颈。义体化人口的维护成本、戴森球的建造与运营成本、星际基础设施的运转成本——所有消耗叠加后，超过了老年星系的自然能源补充速度。

**戴森球的陷阱**：戴森球截获了恒星的全部辐射，短期内提供了近乎无限的能源，掩盖了危机；但恒星寿命有限，建造戴森球消耗了大量行星物质，且义体化文明的需求随技术进步持续膨胀，最终资产负债表彻底失衡。

**无法"减量"的困境**：数十亿义体化个体的硅基躯体和电子脑需要持续供能才能维持生命，不能简单"关机"。主战派正是以此为据：要求全体"降低消耗"等同于要求大规模死亡，不如主动出击争取资源。

## 五、 求救信号与地球

**信号发出**：伽南危机进入不可逆阶段后，塔伦文明向宇宙广播求救信号。

**地球回应的真相**：浩瀚宇宙中，只有地球做出了应答——并非官方决策，而是一名地球研究员出于个人好奇心随手发出的应答信号。他不知道对方是谁，也不知道自己做了什么。

**发现真相后的绝望**：探测确认地球是新生文明，无法提供技术援助；但地球所在太阳系物质与能源资源完好，理论上可延续塔伦文明数千年。

## 六、 社会撕裂：两派论战

### 主战派
- 文明若不能存续，一切都是空谈
- 地球星系资源是当前唯一可及的出路
- 若等待地球文明进入高速发展期，己方优势将消失，窗口彻底关闭
- 义体化人口无法"降低消耗"式地活着，只有扩张才是出路

### 主和派
- 即便掠夺地球星系全部资源，也只能延续数千年，危机本质未变
- 文明的发展模式才是病根；不改变模式，迟早重蹈覆辙
- 他们研究过自己完整的文明档案，知道义体化扩张这条路通向哪里
- 掠夺一个无辜的新生文明在道义上不可接受

> **核心悲剧**：主和派在逻辑上完全正确，但绝望不允许冷静。

## 七、 内战与政变结果

主战派发动政变，双方进入全面内战，持续多年。

**结果**：
- 主战派取得全球政权
- **主和派高层大部分被投入监狱，并未处决**——思想与人员均有保留
- 主和派基层力量被分散压制，失去组织能力，但未被彻底清洗

**主战派保留主和派高层的可能原因**：
- 政治计算：处决会制造殉道者，监禁更彻底地剥夺其影响力
- 留作后手：若征服地球失败，主和派的"可持续发展"方案可能是最后的备选

内战本身对已经资源枯竭的文明造成了额外的毁灭性消耗——主战派赢得政权时，整个文明已千疮百孔。

## 八、 跃迁与抵达地球

内战结束后，主战派动员全星最后的可用资源完成跨星系跃迁。

**抵达时的真实状态**：
- 经历母星危机与内战双重消耗，军事力量严重折损
- 跃迁耗尽了最后的战略储备
- 必须保持地球物质资源的完整性（不能轨道清场），只能打地面战争
- 无法以绝对优势速胜，陷入三十年消耗战

## 十、 叙事含义

**镜像悲剧**：塔伦人是人类的未来——走过相似的进化路线，实现了人类正在梦想的一切（戴森球、义体永生、星际跃迁），然后被它毁掉了。他们看着地球，看到的是几十亿年前的自己。

**主和派高层仍在狱中**：这是最重要的开放伏笔。林默穿越改变历史，操作空间不只是"让人类更早造出核弹"，还包括：干预塔伦内战结果，让主和派的方案得以实施——让两个文明都找到出路。

## 十一、 军事装备：战术背包（生命维持型）

**外观**：标准制式战术背包，部分结构通过硬件接口直接插入士兵背部的义体接驳端口，与血液循环系统物理连接。

**核心系统**：内置微循环血液透析泵，持续向体内注入维持液。

**维持液成分**：

| 成分 | 作用 |
|------|------|
| 高浓度葡萄糖 | 满足无肌肉抑制素体型的极高基础代谢需求 |
| 母星草药提取物 | 功效类似地球红景天，适应原类——缓解塔伦人在地球环境中的"高原反应"：异星大气成分、重力、辐射频谱与伽南存在差异，导致生理应激 |
| 维生素与矿物质复合剂 | 补充地球环境中无法自然摄取的伽南特有微量元素 |

**战场含义**：

- **接口即要害**：背包与身体硬连接，破坏背包等同于直接伤害士兵的血液循环系统；人类一旦掌握这一弱点，针对背包的战术远比正面击杀更具效率
- **库存倒计时**：母星草药无法在地球合成，跃迁时携带的存量是最后的储备；三十年消耗后库存已极度紧张
- **替代品研究**：驻地医疗部门据悉已在尝试以地球植物（包括红景天）替代部分母星草药成分，但效果存疑

**人类的意外食物来源**：

两个走过相似进化路线的碳基物种，基础营养需求高度重叠。维持液稀释后对人类同样具备适口性与营养价值——葡萄糖直接可用，适应原类草药提取物对人类同样有效，矿物质与维生素元素跨物种普适。

在地表粮食体系近乎崩溃的2056年，缴获背包是重要的求生手段，但有明显局限：

- **高糖腐蚀**：长期饮用高浓度葡萄糖溶液对牙齿腐蚀严重，长期依赖者普遍有明显的龋齿与牙釉质损坏
- **营养单一**：成分以糖、草药提取物、矿物质为主，缺乏蛋白质与脂肪，无法作为完整饮食长期维持
- **口感差**：味道介于苦涩草药汁与糖水之间，适口性勉强，难以长期接受
- **定位**：求生口粮而非战略物资——能防止饿死，但以此为主食的人群健康状况持续恶化；地下掩体中依赖背包度日的幸存者，往往牙口极差、体型消瘦
