# 服装设计约束

## 总原则
- 先身份，再功能，再美感
- 先时代，再材质，再细节
- 先稳定，再变体

## 应追求的效果
- 宋代服装结构清楚
- 材质真实，能看出麻、皮、布、金属、木件差异
- 磨损与使用痕迹可信
- 和角色体态、行动方式匹配

## 应避免的效果
- 古偶服化道模板
- 仙侠化披风与飘带
- 不合时宜的高饱和度刺绣
- 过于干净的新衣感
- 游戏皮肤式的视觉堆料

## 角色方向

### 陆晓晓
- 山村求生期：粗麻、旧皮、草绳、木钗、寒地磨损
- 不要画成贵女落难妆
- 不要给她夸张裙摆和装饰性长袖

### 陆辰 / 林默
- 初期：功能性、粗粝、耐打、便于活动
- 不要贵公子侠客服
- 不要现代战术服直接穿越到宋代

### 军营角色
- 层次分明但不过度统一
- 要体现军营旧耗和补丁感

## 阶段化变化
- 角色换装必须基于剧情推进
- 变的是外层与身份标识
- 不变的是体态、功能逻辑和核心识别元素
