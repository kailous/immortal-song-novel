# 小说创作控制 Skill (Novel Creator)

本 Skill 为《大宋工业帝国与星海反击战》项目提供结构化的创作流程，确保“硬核科幻 + 重工大宋”的基调始终如一。

## 核心理念：“重工大宋，星海怒火”
- **风格设定**：严禁出现纯粹的“魔法”或“武侠”套路。所有技术优势必须源自主角的 2056 年知识、外星手环（有限资源）或大宋的工业潜力。
- **基调**：冷峻、宏大、逻辑严密。从 12 世纪社会向星际文明的跃迁必须通过“技术复利的恐怖力量”合理实现。
- **语言**：融合宋代古风谈吐（对话与氛围）与精准的工业/技术术语。

## 创作流程阶段

### 第一阶段：设定沉浸与同步（背景校验）
在开始任何新章节或大纲细化前：
1. **同步设定与日志 (Log Sync)**：阅读 `设定库/00_全局时间线.md` 以获悉正文的**真实剧情进度**。阅读相关角色的 `生平志` 以对齐其个人时空状态。
2. **强制规则规约**：查阅 [设定库指导手册.md](file:///Users/lipeng/Documents/Repository/immortal-song-novel/%E8%AE%BE%E5%AE%9A%E5%BA%93/%E8%AE%BE%E5%AE%9A%E5%BA%93%E6%8C%87%E5%AF%BC%E6%89%8B%E5%86%8C.md) 以确保记录、更新及调取的合规性。
3. **智囊咨询（建议与参考）**：针对技术细节或地缘政治，查阅 `智囊团/` 中的专家意见。智囊团仅提供建议与参考。
4. **手环边界**：务必确认外星手环的当前状态与限制（参考 `设定库/01_底层逻辑与物理机制/` 下的相关条目）。

### 第二阶段：大纲拆解与命运锁定（总监策划）
1. **司马迁命运锁定**：在拆解具体章节前，由 **总策划（司马迁）** 设定本卷/本章的“兴衰曲线”，识别并埋下必要的反转与伏笔，坚决杜绝“直捣黄龙”的平铺叙事。
2. **卷级转章级**：在总监把控的节奏下，将 `大纲/` 中的高层卷大纲拆解为具象的、富有张力的“细纲”（Chapter Outline）。
3. **逻辑预检**：使用 `scripts/context_manager.py` 在动笔前识别任何潜在的设定冲突。

### 第三阶段：草稿撰写（写手执笔）
1. **氛围营造**：强调冲突意象：临安城的蒸汽机、配有燧发枪的重甲骑兵、以及龙脉之巅的“超空间引力波广播矩阵”。
2. **写手协作（节奏与细节）**：由 **写手（如刘慈欣、曹雪芹）** 负责把控剧情节奏、优化文笔细节。**刘慈欣**负责硬核脑洞与宏大场面，**曹雪芹**负责社会礼法与情感深度。
3. **角色语调**：维持女帝（威严+远见）与主角（生存专家+文明导师）的独特语境。

### 第四阶段：同步一致性与质量控制
1. **日志实时对齐 (Log Update)**：章节定稿后，**必须立即**更新以下内容：
    - `设定库/00_全局时间线.md`：记录本章发生的宏观世界变动。
    - 各出演角色的 `生平志`：记录本章内的个人路径节点（章节 LOG）。
2. **技术审查**：科学逻辑是否足够“硬”？是否遵循了世界政府时期的“零内战”铁律？
3. **网站发布更新 (Sync to Web)**：章节定稿并同步日志后，运行 `python3 .agents/skills/novel_creator/scripts/publish_to_docs.py`。该脚本会自动将 Markdown 转换为项目网站所需的 JSON 格式并更新章节索引。

## 常用工具指令
- `python3 .agents/skills/novel_creator/scripts/context_manager.py --summary`：生成当前项目设定的简要报告。
- `python3 .agents/skills/novel_creator/scripts/context_manager.py --check`：扫描正文与设定库之间的不一致项。
- `python3 .agents/skills/novel_creator/scripts/publish_to_docs.py`：同步正文至 GitHub Pages 网站目录。

## 行为准则
- **日志逻辑 (Logs vs Plans)**：严禁在 `设定库/` 中记录尚未在正文发生的未来规划。所有的规划（Roadmap）、命运预设、战役路线图必须存放于 `大纲/` 目录。
- **去系统化**：绝对禁止引入任何“系统加点”或“系统现成图纸”。技术的突破必须体现出“概念-工匠研发-失败-定型”的工业过程。
- **敌方智能**：在构思敌方行动时，优先考虑其对大宋技术的策反、窃取与反制逻辑。
- **神级角色禁止**：主角是一个会犯错、会疲惫、会面对惨烈伤亡而自我怀疑的凡人兵王。
- **碳基变异限度**：子嗣的异能必须严格遵循碳基生物极限，严禁出现超自然超能力。
- **仁慈统一与智者归心**：优先利用技术输出解决生存压力。在描写敌方领袖时，强调其面对文明落差时的心理挣扎。
- **内部肃清与亲子审判**：当后方出现贪腐和特权（特别是主角子嗣）时，主角必须展现绝对的冷酷与法制。通过“全球直播大审判”等极端冲突，确立文明法则凌驾于血亲之上的铁律。
- **NEVER** 引入魔法或超自然元素。
- **ALWAYS** 在章节定稿前执行设定库校验和网站发布同步。
