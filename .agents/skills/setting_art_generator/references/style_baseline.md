# 公开设定图视觉基线

基于当前 `docs/images/characters/*.webp` 的公开设定图整理。

## 统一规律
- **统一画幅**：全部为 `1:1` 正方形
- **统一用途**：图鉴/档案图，不承担剧情叙事
- **统一构图**：主体居中，单体为主；种族图可双体并列，但仍然居中
- **统一背景**：深色、低干扰、轻环境感，不做复杂战场叙事
- **统一信息层级**：先看清对象，再看材质，再看细节
- **统一风格**：写实、冷峻、克制、电影级材质，但不做海报化排版
- **统一输出形式**：纯图像，不直接把说明文字、参数表、边框 UI 烧进图里

## 各类对象的表现规律

### 人物
- 半身到全身之间，正面或近正面
- 表情克制，直接建立脸部基线
- 服装完整、轮廓清楚、背景不抢戏

### 道具
- 单物件独立展示
- 轻透视，不夸张
- 材质和表面纹理优先于氛围

### 种族
- 优先讲清结构差异
- 如果单个个体不足以说明设定，可以双体并列
- 双体并列时仍然要保持“档案式陈列”，不是剧情场景

## 反推重绘时的硬规则
- 不要默认横图
- 不要默认剧情海报
- 不要默认信息图板、说明文字、标注框
- 不要加入大段文字、logo、封面设计
- 不要为了“更酷”牺牲结构清楚度
- 如果现有公开图是官方基线，重绘必须解释修正理由
