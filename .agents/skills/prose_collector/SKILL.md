# 范文收集 Skill

## 适用场景

当用户要求补充范文库、按写法搜集参考段落、或围绕某种叙事问题建立案例库时使用本 Skill。

**典型请求：**
- 补充战斗场面范文
- 搜集情感克制写法
- 参考某位作者的历史氛围
- 为当前章节找可借鉴的对白节奏样本

---

## 执行流程

### 第一步：读取现有库（避免重复）

读取 `资料/02_范文/` 目录下已有文件，明确现有收录情况，确定本次补充方向。

### 第二步：联网搜集

根据用户指定的场景类型或作者，使用 WebSearch 搜集以下优先来源：

**优先作者（高口碑）：**
- 猫腻（《庆余年》《将夜》《间客》《择天记》）— 战斗节奏、情感克制
- 马伯庸（《长安十二时辰》《风起陇西》《显微镜下的大明》）— 历史氛围、底层视角
- 余华（《活着》《许三观卖血记》）— 极简笔法、情感力量
- 陈忠实（《白鹿原》）— 厚重质感、男性气场
- 路遥（《平凡的世界》）— 底层情感克制
- 阿城（《棋王》《树王》）— 极简汉语的极致
- 莫言（《红高粱》）— 感官密度、暴力美学
- 古龙（后期作品）— 对话留白、短句节奏

**搜索关键词格式：**
```
"[作者名] [作品名] 经典段落"
"[作者名] 文笔赏析 [场景类型]"
"[场景类型] 小说写法 范文"
```

### 第三步：筛选标准

收录段落**必须满足**以下至少两条：
- [ ] 动作后不立刻自我解释
- [ ] 有参差的段落节奏（短段+长段）
- [ ] 用行为/感知传递情感，不直说
- [ ] 有一处让读者比角色先看明白的时刻
- [ ] 历史感来自具体细节而非形容词堆叠

**坚决不收录：**
- 大段心理独白后接"原来如此"式感慨
- 战斗中穿插爱国口号
- 对称句式超过三组的段落
- 带"某种说不清的感觉"类模糊套词

### 第四步：整理格式

每条范文的标准格式：

```markdown
### [编号]《作品名》· [场景标题]（作者）

> [原文段落]
> （多行保持引用格式）

**技法**：[50字以内，点明可借鉴的具体手法，直接说"适用于本书XX场景"]
```

### 第五步：保存

**新场景类型** → 新建文件 `资料/02_范文/优秀范文_[场景类型].md`

**已有场景** → 追加到现有对应文件末尾，并更新文件内"速查表"

**同时更新** `资料/00_索引/记忆宫殿.md` 的对应索引条目（如有）。

### 第六步：提交推送

```bash
git add 资料/02_范文/
git commit -m "补充范文库：[场景类型]（[来源作品]）"
git push
```

## 注意事项

- 原文段落如无法核实，注明"基于原著文风高度还原，建议以正版原书核对"
- 不得整章大段收录，每段控制在 200 字以内
- 点评必须联系本书具体场景（陆辰/晓晓/宋兵），不做脱离语境的文学鉴赏
- 优先补充“当前写作确实会用到”的范文，不做无目标堆库
