# 16:9 海报模板规范

当前默认模板采用 **B 方案优化版**，并以 `资源/品牌/poster-template-overlay.png` 作为固定透明叠层模板。后续生成优先通过脚本直接叠加该模板，而不是临时自由摆放。

## 默认布局
- 画幅：`16:9`
- 主视觉区：占画面约 85% 以上
- logo：优先使用官方单色版 / 反白版 / 仅文字极简版
- logo 位置：默认固定为右下角小 Logo 信息组
- 章节标题：如已使用完整小说 logo，默认不再额外叠加大标题
- 短引语：仅在用户明确要求时加入，且必须低干扰
- 安全区：右下角必须保留模板级信息安全区，顶部和左侧保留画面呼吸空间

## 设计目标
- 不干扰主画面
- 一眼看起来像电影剧照壁纸，不像传统网文海报
- 信息层级极少，阅读路径明确
- 适合后续所有插图统一套版
- logo 必须像正式商业 VI 延展，不像临时贴上的角标

## 元素规则

### 1. Logo
- 默认从官方版式中选用以下版本：
  - 深色背景：`03 纯白反白版`
  - 浅色背景：`02 纯黑单色版`
  - 画面信息已很满时：`06 仅文字无纹理极简版`
- 非明确要求下，不使用厚重金属浮雕版做章节插图落版
- 不使用高饱和发光描边
- logo 是品牌识别层，不是标题装饰层
- logo 默认宽度控制在画面宽度的 `12%–22%`
- 除非是专门的标题海报，不得把 logo 放到画面中心主视觉位

### 1.5 Logo 摆放优先级
- 第一优先：右下角小 Logo 信息组，严格参照当前 B 版模板
- 第二优先：仅当用户明确要求换模板时，才讨论其他布局
- 若人物、武器、手部、脸部或关键道具靠近右下安全区，应先调整镜头和人物站位，而不是改模板

### 2. 文案
- 章节标题采用细字重、较松字距
- 短引语不超过 1–2 行
- 文案整体透明度和对比度要克制，不能压过画面主体
- 若完整 logo 已包含书名，章节标题应退为副信息，不能与 logo 形成双标题冲突

### 3. 背景处理
- 允许在底部或角落信息区加入轻微暗部渐层，提升字标可读性
- 可使用极轻的颗粒、极细分隔线
- 不允许大边框、重装饰、复杂金属纹、过度 UI 化面板

## 一致性要求
- 所有后续海报默认先按本模板排版
- 除非用户明确要求，不能临时切换成别的模板方案
- 所有出图先判断该画面适合 `02 / 03 / 06` 中哪一版 logo，再决定位置
- 若画面本身不适合右下角信息区，优先调整构图和人物调度，不先推翻模板
- 若上传了新的官方 VI 参考图，以最新参考图覆盖旧经验判断

## 失败特征
- logo 太大
- logo 版本和官方 VI 不一致
- 没有遵守右下角小 Logo 信息组
- 白字压住人物脸或关键动作
- logo 像临时贴上的角标，而不是正式商业字标
- 信息区像传统网文封面角标
- 章节标题和引语太重，破坏电影剧照感
- 为了“设计感”加了过多边框、金属件、装饰纹样
