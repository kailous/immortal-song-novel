# 16:9 海报模板规范

当前默认模板采用 **B 方案优化版**，方向参考高端电影剧照壁纸与流媒体宣传剧照。

## 默认布局
- 画幅：`16:9`
- 主视觉区：占画面约 85% 以上
- logo：右下角小尺寸反白纯白版
- 章节标题：logo 上方或相邻位置，极细字重，低干扰
- 短引语：与章节标题同组排布，优先单行或短双行
- 安全区：右下角保留稳定边距，避免贴边

## 设计目标
- 不干扰主画面
- 一眼看起来像电影剧照壁纸，不像传统网文海报
- 信息层级极少，阅读路径明确
- 适合后续所有插图统一套版

## 元素规则

### 1. Logo
- 默认使用纯白反白版
- 不使用厚重金属浮雕效果
- 不使用高饱和发光描边
- 尺寸要小，处于辅助识别位，不抢画面主体

### 2. 文案
- 章节标题采用细字重、较松字距
- 短引语不超过 1–2 行
- 文案整体透明度和对比度要克制，不能压过画面主体

### 3. 背景处理
- 允许在右下信息区加入轻微暗部渐层，提升白字可读性
- 可使用极轻的颗粒、极细分隔线
- 不允许大边框、重装饰、复杂金属纹、过度 UI 化面板

## 一致性要求
- 所有后续海报默认先按本模板排版
- 除非用户明确要求，不能临时切换成别的模板方案
- 若画面本身不适合右下信息区，先微调安全区，不先推翻模板

## 失败特征
- logo 太大
- 白字压住人物脸或关键动作
- 信息区像传统网文封面角标
- 章节标题和引语太重，破坏电影剧照感
- 为了“设计感”加了过多边框、金属件、装饰纹样
