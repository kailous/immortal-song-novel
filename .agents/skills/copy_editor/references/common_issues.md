# 常见写作问题清单

用于正文、设定、图鉴、词典的快速审校。优先检查以下高频问题：

## 1. 中文排版
- 中文正文中误用英文双引号 `"..."`。
- 拟声、绰号、引语没有统一成 `“……”`。
- 破折号、省略号、书名号前后风格不一致。

## 2. 固定搭配和常用词
- 固定搭配写反，如“走兽飞禽”应优先检查是否应为“飞禽走兽”。
- 力度动词搭配不当，如“捏死了扁担”这类不自然表达。
- 身体部位表达生硬，如“一侧脖筋”优先检查是否应改为“颈侧”。

## 3. 病句与句法松散
- 插入语过多，导致句子主干发虚。
- 数量表达含混，如“两三个分量”这类不明确写法。
- 重复并列句式过密，如连续“看……；看……”。

## 4. 用词轻重不匹配
- 场面强度很重时，动词却偏轻，如“沾着鲜血”。
- 血肉、伤口、坠落等场景里，避免使用不够贴身的轻飘词。

## 5. 修辞和叙事协调
- 修辞过满但动作信息不足。
- 句子有气氛，但定位不够清楚，读者要二次解析才明白主干。
- 同一句里同时追求古意、画面和解释，导致负担过重。

## 6. 替换优先级
遇到问题时，优先按以下顺序修：
1. 明确错别字和标点
2. 明显病句
3. 搭配错误
4. 数量词、部位词、动作词不准
5. 轻度赘述和重复

## 7. 不要过度修
- 如果一句只是风格化，而不是语病，不要硬改。
- 如果问题需要整段重写，不要伪装成“替换一个词”。
- 审校优先保留原文气质，只修真正影响阅读的点。
