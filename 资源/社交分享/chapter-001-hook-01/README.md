# 第一章获客视频包 · Hook 01

## 主钩子
**末日工程师被撕成分子后，掉进南宋；他左手的黑色手环里，装着人类最后的科技。**

这是当前第一章最容易获客的钩子，原因很直接：
- 有强反差：末日科幻 + 南宋乱世
- 有强问题：一个现代人为什么会掉进南宋
- 有强道具：黑色手环天然适合做视觉记忆点
- 有强后续承诺：科技树会被点歪

## 推荐投放方向
- 抖音：先打“设定反差”和“命运瞬间”
- 小红书：先打“电影感封面 + 高概念文案”

## 目录内容
- `video-prompts.md`：短视频文案、分镜、生成提示词和字幕口播
- `assets/`：本方案专用素材拷贝

## 推荐使用顺序
1. 先用 `assets/cover-portrait.png` 或 `assets/douyin-share-1080x1920.png` 做封面
2. 再用 `video-prompts.md` 的 15 秒版生成视频
3. 第一条跑通后，再扩展到 30 秒版
