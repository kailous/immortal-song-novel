# 社交分享素材包

当前目录用于归档《长生不死》的短视频、封面和平台传播素材。

## 当前结构

- `预告片/`：当前正在使用的预告片方案目录。
- `预告片/预告片 v1.0.mp4`：当前整条预告片成片。
- `预告片/分镜头/`：逐镜头导出稿，便于二次剪辑或替换。
- `预告片/分镜头/分享/`：分镜头静态分享图，适合单张发帖或做轮播。

## 后续目录约定

- `预告片/`：整条预告片或预告片迭代版本。
- `chapter-xxx-hook-yy/`：按章节和钩子归档的专项获客素材包。
- 每个方案目录至少包含：
  - `README.md`：说明定位、内容和使用顺序
  - `video-prompts.md`：文案、分镜、字幕、生成提示词
  - 方案所需最小素材集

## 使用原则

- 一个目录只服务一个明确传播目标，不混放不同用途的素材。
- 优先保留正式导出稿，不在这里堆叠过多中间废稿。
- 需要长期复用的品牌素材继续放在 `资源/品牌/`，这里只保留该方案实际用到的版本。
- 新方案优先新建子目录，不直接覆盖现有方案，便于回看投放结果。
