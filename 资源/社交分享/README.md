# 社交分享素材包

当前目录用于归档短视频和社交平台获客素材包。

## 目录约定
- `chapter-xxx-hook-yy/`：按章节和钩子方案归档。
- `assets/`：实际出图、剪辑、封面和品牌素材拷贝。
- `README.md`：该方案的定位、钩子和使用顺序。
- `video-prompts.md`：视频生成提示词、分镜和字幕建议。

## 使用原则
- 一个目录只服务一个明确钩子，避免把多种卖点混在一起。
- `assets/` 里只放该方案需要的最小素材集，不做全量镜像。
- 优先使用 `资源/品牌/` 和 `资源/插图/` 的当前正式版本复制过来，避免剪辑时误拿旧图。
