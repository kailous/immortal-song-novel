# 预告片素材包

当前目录用于归档《长生不死》第一版预告片及其拆分素材。

## 目录内容

- `预告片 v1.0.mp4`：当前成片。
- `分镜头/01-06.png`：逐镜头静帧。
- `分镜头/01-06.mp4`：逐镜头动态片段。
- `分镜头/分享/01-06.png`：适合单独发布的分享图。
- `video-prompts.md`：本方案对应的文案、字幕和生成提示词整理。

## 推荐使用顺序

1. 先看 `预告片 v1.0.mp4`，确认整条节奏是否仍作为当前基线。
2. 需要局部替换时，直接从 `分镜头/` 选对应镜头重做，不必整条返工。
3. 需要社交平台预热时，优先使用 `分镜头/分享/` 里的单张图做分发。

## 维护约定

- 新版成片继续按 `预告片 vX.Y.mp4` 命名。
- 如果镜头顺序调整，仍保持两位数编号，避免后续脚本或人工排序混乱。
- 只在确认成体系后再补充新子目录，避免把临时草稿和正式稿混在一起。
