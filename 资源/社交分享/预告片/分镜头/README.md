# 预告片分镜头索引

当前目录保存 `预告片 v1.0.mp4` 的逐镜头拆分稿。

## 文件规则

- `01-06.png`：对应镜头静帧基线。
- `01-06.mp4`：对应镜头动态片段。
- `分享/01-06.png`：为社交平台单张发布单独导出的分享图。

## 使用建议

1. 要替换成片里的某一段时，优先只改对应编号，不整条返工。
2. 要做短版混剪时，先从 `01`、`03`、`04`、`05` 里选强节奏镜头。
3. 要做图文预热时，优先看 `分享/` 下的导出图，不直接从原静帧里临时裁。
