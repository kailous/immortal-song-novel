# IP 与非商用使用政策

本文件适用于《长生不死的我，在南宋点歪了科技树》项目中的小说内容、设定文本、角色形象、插图、图鉴、世界观资料、品牌视觉与其他创作资产。

## 1. 默认立场

除权利人另行书面授权外：

- 所有核心内容与资产均保留版权
- 仓库公开不等于放弃权利
- 技术框架可开放复用，不等于角色 IP、品牌 IP、世界观 IP 可以自由商用
- 内容资产默认按 `CC BY-NC-SA 4.0` 进行非商用共享与二创

## 2. 允许的非商用使用

在满足署名、保留出处、不暗示官方认可的前提下，允许：

- 个人阅读、学习、研究
- 非商用引用、评述、书评、资料讨论
- 非商用分享仓库链接或站点链接
- 非商用二创、改写、设定延展、翻译、同人讨论
- 非商用前提下对内容进行再发布，但必须说明是否修改过，并继续按相同的非商用规则分发

## 3. 需要事先书面许可的行为

以下行为默认不被允许：

- 任何商业使用
- 任何商业改编或商业衍生开发
- 未经许可的大规模转载正文、图鉴、设定或图片
- 以本项目为基础制作并销售游戏、影视、漫画、动画、广播剧、有声书、周边或课程
- 将本项目文本、图片、设定、角色或品牌资产用于商业 AI 训练、数据集整理、模型微调、商业代理产品或商业检索库
- 使用本项目角色、标题、Logo、世界观设定进行招商、联名、代言、推广或营销包装
- 冒充官方版本、官方账号、官方授权项目

## 4. 角色与世界观 IP 保护

以下元素不因仓库公开而进入公共领域，也不因默认非商用规则而自动获得衍生授权：

- 作品名与系列名
- 角色名、组织名、种族名、道具名
- 角色形象、关系设定、代表性视觉元素
- 世界观结构、核心设定组合、图鉴体系
- 官方宣传语、品牌设计、封面与主视觉

## 5. 禁止的滥用方式

尤其禁止以下滥用：

- 拿角色和设定直接做商业号、商业账号包装或商业内容矩阵
- 将官方图片、角色图鉴、封面、Logo 改成“二创官方版”继续传播
- 用项目名称、角色名称或官方视觉做误导性宣传
- 将仓库内容打包成“免费资料”“提示词包”“世界观素材包”进行售卖或引流
- 未经许可，把本项目作为商业模型训练语料或图像训练集的一部分

## 6. 署名与出处

在允许范围内使用时，至少应：

- 标注作品名
- 标注原始仓库或原始发布页出处
- 标注是否做过修改
- 不得暗示作者对该使用行为进行了认可、合作或背书
- 若继续分发二创内容，应维持非商用共享原则，不得转成闭源商业售卖包

## 7. 保留权利

权利人保留：

- 追究侵权、假冒、误导性使用和未经许可商用的权利
- 单独授予商业改编、转载、联名、出版、发行与训练授权的权利
- 随时对官方品牌资产、人物设定图、宣传图、封面资产作进一步限制说明的权利
