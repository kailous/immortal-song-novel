# Agent 操作日志

用途：记录 agent 已执行的实际动作、改动范围、命令链路和提交结果，供事后追溯与 Git 历史对照使用。

使用约定：

- 这是**追溯日志**，不是日常上下文入口。
- 平时获取项目上下文时，默认**不检索、不优先读取**本文件。
- 只有在需要回查“某次到底做了什么”“某个提交为什么产生”“最近变更链路是什么”时，才读取本文件。
- 正常创作、出图、发布时，优先读取 `当前状态.md`、`铁律速查.md`、相关 skill 和目标文件本身。

