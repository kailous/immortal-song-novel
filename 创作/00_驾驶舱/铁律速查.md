## A. 世界观铁律（不可突破）

| # | 铁律 |
|---|------|
| A1 | **零魔法**：技术优势只来自①2056年知识 ②外星手环（有限资源）③大宋工业潜力。 |
| A2 | **零系统**：无属性面板、无任务日志、无技能列表、无系统弹窗、无"目标达成"提示。 |
| A3 | **工业过程必须完整**：技术突破必须经历"概念 → 工匠研发 → 失败 → 定型"，不可一键造物。 |

| A5 | **主角是凡人兵王**：会犯错、会疲惫、会自我怀疑，严禁神化。 |
| A6 | **子嗣异能在碳基极限内**：严禁超自然超能力。量子重置能力绝对无法遗传。 |
| A7 | **设定只记 Log**：只写已发生事实，未来规划一律放 `创作/01_大纲/`。 |

## B. 叙事表达禁区

**禁止出现的词汇 / 结构：**

- 视网膜字符、幽绿色光幕、进度条、血条、威胁评级（作为系统弹窗）
- `判定：结果` / `目标：描述` 这种日志式冒号短语
- "社会学契约"、"接入点"、"任务提示"、"目标达成"

**推荐替代表达：**

- 战术直觉 / 肌肉记忆 / 空间覆盖 / 生理闭环 / 血色支点
- 战场感知用"气流、心跳、杀气的感知"代替 HUD

## C. 对白语调速查

| 角色 | 语调 |
|------|------|
| **陆辰** | 极简精准 + 中原/河南方言底色（中、弄啥、寻思、信球、没屁搁那儿楞搁） |
| **陆晓晓** | 宋代官话（临安雅音）；亲昵时露淡淡吴语韵味 |
| **李德海及基层军官** | 军旅官话：粗放实用（身头、差事、王师） |
| **村民/土匪/游民** | 风格化吴语（晓得、弗晓得、作对、弗利索、细人） |

> 陆辰的中原口音在南宋人耳中是"汴京旧都雅言"，背书其"将门之后"身份。

## D. 会议制度触发词

| 触发词 | 动作 |
|--------|------|
| `"开始会议"` | 进入论证期：只讨论，锁定所有文件修改，实时更新 `创作/04_会议/纪要/` |
| `"继续会议 [文件路径]"` | 从已有会议记录继续讨论 |
| `"会议结束 执行"` | 进入执行期：按决议批量修改设定 / 公开正文 / 大纲 |
