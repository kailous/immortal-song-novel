# 新书项目 (New Novel Project)

这是新小说的项目根目录。

## 目录结构说明

- `大纲/`: 存放整体大纲、分卷大纲、剧情细纲等。
- `正文/`: 存放小说正文的草稿和修订版。
- `设定库/`: 存放世界观、角色设定、时间线、势力设定等内容。
- `参考资料/`: 收集的写作素材和灵感记录。
- `会议记录/`: 讨论、头脑风暴记录。
- `写手/`: 合作写手的文档、排班、交流记录。
- `交付/`: 整理出稿、用于发布的最终文件。

## 配置与脚本
- `book_info.json`: 书籍的基本元数据信息，如书名、作者、UUID。
- `build_epub.py` / `generate_base64_cover.py`: 自动生成和构建 EPUB 的脚本。
