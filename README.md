# 长生不死 · 仓库说明

仓库现在按三层组织：

- `创作/`：内部创作中台。包含驾驶舱、大纲、设定、团队档案和会议纪要。
- `资料/`：只读资料库。分为索引、原典、范文、研究四层。
- `docs/`：对外网站与公开内容。章节和图鉴的唯一发布源都在这里。

## 关键入口

- `创作/00_驾驶舱/`：每次创作会话的唯一入口。
- `创作/01_大纲/`：未来规划，只存未发生内容。
- `创作/02_设定/`：已发生事实与内部规则，只存 Log 和底层设定。
- `创作/03_团队/`：写手、智囊团、联审材料。
- `创作/04_会议/纪要/`：正式会议存档。
- `资料/00_索引/记忆宫殿.md`：查资料前先读的总索引。
- `资料/01_原典/`：原始文献与论文。
- `资料/02_范文/`：写法参考。
- `资料/03_研究/`：整理后的研究材料。
- `资源/`：内部源图与参考图。

## 发布链路

- 中文正文源：`docs/content/chapters/zh/`
- 英文正文源：`docs/content/chapters/en/`
- 中文图鉴源：`docs/content/profiles/zh/`
- 英文图鉴源：`docs/content/profiles/en/`
- 发布命令：`make publish`、`make sync-chars`、`make audit`
