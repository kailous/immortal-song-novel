# 技术殖民与和平演变
> 仁慈统一准则 · 降维施舍策略

## 核心哲学：生存权的降维施舍
主角深知古代战争的本质是“存量资源的零和博弈”。他利用2056年的技术包，将博弈从“抢夺粮食”翻转为“共同创造增量”。

## 「技术换和平」策略
**生存保障三件套**：
- **高产作物**：通过选育或手环干预引入土豆、玉米等高热量作物，解决寒旱地区饥荒。
- **能源革命**：推广煤炭蜂窝煤与简易火炉，降低北方严冬的死亡率。
- **现代医学**：初级抗生素（土法提纯）与防疫体系对异族开放，建立“医者无疆”的文明神话。

**临时自治区（TAZ）**：允许部分敌对政权在一定期限内保留文化惯习，但自治区内必须普及大宋语教学、接受大宋银行货币体系。通过经济纽带实现事实上的同化。

## 「智者归心」：面向敌方精英的统战
并非所有敌人都是反派。对真正具有胆识、心系苍生的异族领袖，主角采取“文明共治”模式——邀请敌方领袖亲历大宋工业城，体验产房、学校和不夜城的震撼，用“降维的生活水平”摧毁其执迷。

**战争的真正目标**：火炮倾泻在那些紧抱旧制度烂摊子、宁愿让百姓陪葬也不放权的既得利益者（旧王室、祭司、奴隶主）身上。大宋的军队往往作为“解放者”进入敌方基层。

## 策略的副作用
- **技术扩散风险**：民生技术的普及不可避免地增强了潜在对手的国力。
- **内部反对派**：大宋激进派认为主角对“蛮夷”过于仁慈、浪费资源。这种内部政治摩擦是主角必须面对的治理挑战。
