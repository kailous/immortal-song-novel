# 守望者「哪吒」
> 皇族隐秘调查官 · 思维并行异能者

## 核心定位
- **代号**：哪吒（Ne-Zha）——寓意“三头六臂”般的并行处理能力。
- **身份**：大宋联邦监察处顶级秘密调查官（Special Auditor），皇族名誉守护者。
- **背景**：作为大一统战争后期出生的幼子，在主角身先士卒的铁血家风下成长，深知皇权的边界。
- **外形**：外表聪慧可爱，年幼时便展现出超越常人的沉稳与冷静，是大宋亿万国民心中的“神圣灵童”。

## 异能：思维并行（左右脑互搏）
**生物学原理**：胼胝体高度发达，左右脑信息交换速率为常人十倍，且具备“独立意识单元”。

**能力表现**：
- **双重观察视角**：面对一个人时，左脑分析其语言逻辑和后台财务数据，右脑精准捕捉微表情与情感波动。
- **思维并行 / 数据预判**：在处理全球数以亿计的政务流通数据中，识别隐藏在巨大官僚机器深处的“癌细胞（Systemic Cancer）”——那些游走在规则之外、被利益集团渗透的腐败节点。

**代价**：大脑时刻处于高负荷并行状态，需要大量睡眠，性格略显孤僻——因为常人的思维在他眼中如同幻灯片般缓慢且透明。

## 主线动机：清洗毒瘤与守护家族名誉
他常年处于隐身状态，在星海官僚体系中物理清除那些企图通过“皇权崇拜”进行政治投机的买办与奸臣。

在极少数情况下，他会利用“监察侦查权”，在死刑签署前对某个位高权重的兄长产生心理动摇时提前介入。

他唯一的愿望，就是通过确保制度的无限透明，让父母不再有机会面对那种“神人相杀”的极端悲剧。
